Требуется создать консольный файловый менеджер начального уровня, 
который покрывает минимальный набор функционала по работе с файлами.

1. Функции и требования:
2. Просмотр файловой структуры
3. Поддержка копирование файлов, каталогов
4. Поддержка удаление файлов, каталогов
5. Получение информации о размерах, системных атрибутов файла, каталога
6. Вывод файловой структуры должен быть постраничным
7. В конфигурационном файле должна быть настройка вывода количества элементов на страницу
8. При выходе должно сохраняться, последнее состояние
9. Должны быть комментарии в коде
10. Должна быть документация к проекту в формате md
11. Приложение должно обрабатывать непредвиденные ситуации (не падать)
12. При успешном выполнение предыдущих пунктов – реализовать сохранение ошибки в текстовом файле
в каталоге errors/random_name_exception.txt

При успешном выполнение предыдущих пунктов – реализовать движение по истории команд (стрелочки вверх, вниз)
Команды должны быть консольного типа, как, например, консоль в Unix или Windows.
Соответственно требуется создать парсер команд, который по минимуму использует стандартные методы по строкам.
